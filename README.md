# MGS ZARYA MAIN

Первая версия сайта проекта ZARYA.

Этот репозиторий содержит стартовую веб-версию и служит базой для дальнейшего развития полноценного сайта. Текущая версия фиксирует исходную структуру, визуальное направление и основные разделы, от которых будет вестись дальнейшая доработка проекта.

## Статус

`v1` — начальная версия сайта.

В дальнейшем на основе этой версии планируется:

- развивать структуру и содержание страниц;
- дорабатывать интерфейс и визуальную подачу;
- расширять функциональность сайта;
- использовать репозиторий как основную точку для следующих итераций.

## Структура проекта

- `index.html` — главная страница;
- `pages/` — дополнительные страницы сайта;
- `css/` — стили проекта;
- `js/` — клиентские скрипты.

## Запуск

Для локального просмотра достаточно открыть `index.html` в браузере.
